# 阶段 1 进展日志：工程骨架与独立环境

状态：等待产品负责人最终验收

开始日期：2026-07-23  

## 阶段目标

创建无需下载人脸模型即可启动、测试和维护的本地 Web 工程，为后续模拟推理、人脸检测和精准换脸提供稳定边界。

## 本阶段范围

- 安全对齐本地与 GitHub 提交历史。
- 冻结 Python、Node.js、包管理器和主要框架版本。
- 在项目目录内创建独立 `.venv`。
- 建立 FastAPI 后端、React/TypeScript/Vite 前端和共享任务契约。
- 建立配置、结构化脱敏日志、测试、格式化和静态检查。
- 提供 Windows 启动与诊断入口。
- 每个小节验证通过后再进入下一小节。

## 延续的强制约束

- 不下载或提交人脸模型权重。
- 不使用真实私人照片作为工程骨架测试资产。
- 默认服务只允许绑定回环地址。
- 不记录图片内容、身份向量、密钥或本机绝对路径。
- 保持严格零付费和本地优先。
- 每次公开发布前扫描敏感内容、大文件和禁止资产。
